HikayeStory

SAGI Finance, kişisel finans takibinin gerçekten özel olabileceğini kanıtlamak için geliştirildi. Çoğu fintech uygulaması verilerinizi reklam gelirine dönüştürür; biz tam tersini yapıyoruz: sıfır veri toplama, sıfır profilleme, sıfır satış.

SAGI Finance was built to prove that personal finance tracking can be genuinely private. Most fintech apps monetise your data through ads; we do the opposite — zero data collection, zero profiling, zero selling.

Uygulama PWA mimarisiyle tasarlandı; hafif, hızlı ve çevrimdışı çalışabilen bir yapı. Ücretsiz katman yalnızca localStorage kullanır. Plus katmanında anonim 16 haneli anahtar ile isteğe bağlı Firestore sync mevcuttur.

The app is architected as a PWA — lightweight, fast, and offline-capable. The free tier uses only localStorage. The Plus tier adds optional Firestore sync via an anonymous 16-digit key.

TeknolojiTechnology

Çerçeve bağımlılığı olmayan saf HTML, CSS ve JavaScript ile inşa edilmiştir. Yükleme süresi milisaniyeler, kurulum sıfır.

Built with vanilla HTML, CSS, and JavaScript — no framework dependency. Load time is milliseconds, setup is zero.
